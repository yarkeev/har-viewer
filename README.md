# har-viewer

![](https://raw.github.com/yarkeev/har-viewer/gh-pages/pics/logo.png)

Utility to view and analyze har file

## Installation

```bash
$ npm install har-viewer -g
```
<br/>
<br/>

```bash
$ har-viewer

Starting up har viewer on http://127.0.0.1:3000
```
